# 🏙️ City Chase - Web Game

ボードゲーム「シティチェイス」のローカルマルチプレイ版Webアプリです。

## 🎮 ゲーム概要

**City Chase** は、1人の容疑者と複数の警察官による非協力型の推理ゲームです。

### ゲームルール

- **プレイヤー**: 容疑者1人 + 警察官1人（2人プレイ）
- **ボード**: 5×5の建物マス + 4×4の交差点
- **ゲーム進行**: 最大11ラウンド

### 容疑者の目標
- **11ラウンド逃げ切る** → 勝利！

### 警察の目標
- **容疑者を発見する** または **追い詰める** → 勝利！

## 🎯 ゲームフロー

### Phase 1: セットアップ
1. 警察がヘリ3つ（赤・青・緑）を交差点に配置
2. 容疑者が隠れる建物を選択
3. ゲーム開始

### Phase 2: ゲーム進行
- **容疑者のターン**: 隣接建物に移動 → 痕跡配置
- **警察のターン**: 各ヘリについて「移動」or「検索」を実行
- 最大11ラウンド

### Phase 3: ゲーム終了
- 容疑者が11ラウンド逃げ切る → 🎉 勝利
- 警察が容疑者を発見 → 🚨 勝利
